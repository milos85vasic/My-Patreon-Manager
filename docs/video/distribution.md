# Video Distribution

## YouTube
1. Create a playlist "My Patreon Manager Course"
2. Upload each module as an unlisted video (public after review)
3. Title format: `My Patreon Manager — Module NN: Title`
4. Description template:
   ```
   Module NN of the My Patreon Manager video course.
   Topics: [list from script]
   Code: https://github.com/milos85vasic/My-Patreon-Manager
   Docs: https://milos85vasic.github.io/My-Patreon-Manager/
   Exercise files: examples/moduleNN/
   ```
5. Tags: patreon, go, golang, automation, content-generation, git
6. Add end card linking to next module

## Vimeo (fallback)
Same metadata. Use "unlisted" privacy until reviewed.

## Companion repo
Link to `examples/` in the main repo. Each module directory contains starter files per the exercise section of the module script.

## Accessibility
- [ ] Captions uploaded (from docs/video/captions/moduleNN.srt)
- [ ] Transcript linked in video description
- [ ] Color contrast verified in slides
- [ ] Audio normalized to -16 LUFS
